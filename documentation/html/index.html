EtaGen

An event trigger generator based on Hilbert-Huang Transform
It is a Python module but the core library of EtaGen is built in C/C++

Dependences

  • Numpy >= 1.10
  • Boost.Python < 1.65 or >= 1.64 with Boost.Numpy

## Install

1 $ python setup.py install --user
